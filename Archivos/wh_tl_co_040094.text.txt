Title - Enhancing Elements on Weapons
text01 - Weapons have Elements and Element Levels. Elements come in varieties like Fire or<br>Light, and their Element Level indicates their potency.
text02 - To raise the Element Level, choose <yellow>weapons of the same rarity and category<c> as<br>materials when Enhancing.
text03 - When using weapons of the same name as materials, <yellow>the Element Level will rise<br>significantly<c>. <br><br>Using a weapon of the same name and Element will cause it to rise even more. The<br>maximum Element Level is fifty, but for Rarity 13★ weapons, the maximum is sixty.
text04 - The higher the Element Level is, the more damage it will do to the enemy. If the<br>enemy is weak to the Element, it will inflict even more damage. The Element is<br>determined when the weapon is obtained, but can be changed using certain items.
