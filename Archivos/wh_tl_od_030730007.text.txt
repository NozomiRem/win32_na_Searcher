text01 - Fused Techniques are Techniques that only Forces<br>and Techters can use.
text02 - To use Fused Techniques, you must have a Rod,<br>Talis, or Wand equipped.
text03 - You also need to set them to your Sub-Palette in order to use them. <br>Note that you cannot set them to your Weapon Slot.
text04 - To use Fused Techniques, you must first damage<br>an enemy with the following attacks in order to<br>fill up the gauge.<br><br>Techniques<br>Class Skill -- Photon Bullet<br>Wand Gear -- Photonic Fury<br>Photon Art -- Heavy Hammer
text05 - Once the gauge is full, the icon will<br>illuminate, alerting you that it can be used.
text06 - By pressing and holding the icon, you can begin<br>charging it. Release the icon once it is fully<br>charged to cast the technique.
text07 - Fused Techniques allow you to use two elemental<br>effects at once.<br>Each element is applied to your Class Skills.
text08 - In order to use Fused Techniques at their fullest<br>potential, it is important to learn both Force and<br>Techter Skills.
